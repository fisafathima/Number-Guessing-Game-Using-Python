🎯 Number Guessing Game (Python)

A simple command-line Number Guessing Game built using Python. The program generates a random number between 1 and 100, and the player must guess the correct number. The game provides hints if the guess is too high or too low and tracks the number of attempts.

## Features

* Random number generation between 1 and 100
* Detects whether the guess is too high or too low
* Counts:

  * Number of high guesses
  * Number of low guesses
  * Total guesses
* Input validation using exception handling
* Play Again option
* Beginner-friendly Python project

## Technologies Used

* Python 3
* Random Module

## How to Run

1. Clone the repository:

```bash
https://github.com/fisafathima/Number-Guessing-Game-Using-Python.git
```

2. Navigate to the project folder:

```bash
cd number-guessing-game
```

3. Run the program:

```bash
python number-guessing-game.py
```
